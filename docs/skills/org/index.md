# Org-Wide Skills

These skills are defined in [`winccoa-tools-pack/apm-org`](https://github.com/winccoa-tools-pack/apm-org)
and are deployed to every repository in the organization via [APM](https://microsoft.github.io/apm/).

After cloning any repository and running `apm install`, your AI agent (GitHub Copilot, Claude, Cursor)
automatically receives these skills as context.

Browse the skills in the sidebar.

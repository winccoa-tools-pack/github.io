**Description:**  
Create a set of GitHub Actions for OA projects.

**Actions:**  
- project actions (register, create, start/ stop)
- start manager (ctrl manager) with project
- Panel linting  
- CTRL linting  
- Panel XML conversion  
- Deployment checks  
- DP schema validation  
- all the ctrlppcheck QGs
- testing actions (ctrl, node, gtest, ctest)
- build-documentation
- package project
- encrypt files

**Acceptance Criteria:**  
- Provides at least 3 reusable actions  
- Works with OA projects in CI  
- Includes example workflows  
- Published on GitHub Marketplace
